# Avatar Default

Tempatkan file avatar default di folder ini:

- `default-male.png` - Avatar default untuk laki-laki
- `default-female.png` - Avatar default untuk perempuan

Format yang didukung: PNG, JPG, JPEG
Ukuran yang disarankan: 200x200px atau lebih besar

Jika file tidak ditemukan, sistem akan menggunakan inisial nama sebagai fallback.
